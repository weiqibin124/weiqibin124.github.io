C语言运算符优先级

发表于 2020-04-04 分类于 C语言基础 Valine：
本文字数： 161 阅读时长 ≈ 1 分钟
关于逻辑、赋值等运算符的优先级问题
优先级 运算符 结合性
1 () 从左到右
2 ! + - ++ – 从右到左（单目的+和-）
3 * / % 从左到右
4 + - 从左到右
5 < <= > >= 从左到右
6 == != 从左到右
7 && 从左到右
8 || 从左到右
9 = += -= *= /= %= 从右到左

优先级：关系运算>逻辑运算>赋值运算

强制转换类型的优先级高于四则运算。
